# full-stack-js-example-18072024

# Project overview
An example application for managing posts.
Built with: 
- ReactJS
- GraphQL
- Apollo
- NodeJS
- PostgreSQL
- TypeORM
  
# Setup instructions
1. Clone the repo
2. Install project dependencies with "pnpm i" @ root directory
3. Start project with "pnpm run start"
